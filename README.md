# C-starter

---

## Complete Visual Summary

```
#include <iostream>     // Get the printing tools
                        // (empty line for readability)
int main() {            // Start of program
    std::cout << "Hello";  // Print "Hello"
    return 0;           // Say "success"
}                       // End of program
```

---

## Memory Trick

| Line | Think of it as |
|------|----------------|
| `#include <iostream>` | "Get the toolbox" |
| `int main()` | "Start here" |
| `{` | "Begin" |
| `std::cout << "Hello";` | "Say Hello" |
| `return 0;` | "All good" |
| `}` | "End" |

---

## One Sentence Summary

**`#include <iostream>` gets the print tool, `int main()` starts the program, `{ }` wraps the code, `std::cout << "Hello";` prints text, and `return 0;` says it worked.**
